### What I learned

In this project I had to use CSS Flexbox to position the elements inside the wrapper

```html
<div class="wrapper"></div>
```

```css
.wrapper {
  display: flex;
  justify-content: center;
  align-items: center;
  flex-direction: column;
}
```
